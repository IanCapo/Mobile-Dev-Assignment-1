SWOT Analysis

S W O T
Strenghts:
  • Takes local conditions like weather and traffic into consideration.
  • Has all the functionality you would expect from an alarm clock.
  • Is only available for ios operating system.
  • Only allows one alarm to be set.
  • Availability for other OS.
  • Setting of several alarms for appointments throughout the day.
  • If other maps/traffic monitoring providers (e.g. Google, TomTom) include an alarm function in their portfolio.
