Technology

React Native

React Native is a framework specialised in mobile development. It is open-source and was created by Facebook.

Expo Framework

Expo is a framework that provides a set of tools to make it possible to develop applications that run natively on different devices as well as the web. It can be used with react native.

Geolocation API + Maps Platform/ TomTom API

Geolocation API allows users to provide their current location to web application. With the google maps platform or the TomTom Developer portal we can utilise that and calculate routes and travel times.

npm package manager

npm is a packager manager for Node.js packages. npm has an abundance of packages that provide developers with existitng solutions that can be adapted.
