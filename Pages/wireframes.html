Selected Wireframes

On this page we will introduce a first idea of how we picuture the application.

Home Screen

When the user opens the app he finds himself on the home or main page, from here the user can add new alarm, switch to the settings tab and disable or enable the geolocation.

Home Screen - Onboarding

When the user opens the app for the first time he finds himself on the main page with guiding tooltips that explain how to use the app.

Add an alam - form

The user fills out the form to add an alarm by filling out the fields in the form. Besides start point and destination the user needs to specify time of arrival, time to get ready and whether it is a repeated alarm.

Home Screen - Overview Alarms

Once an alarm is created it appears on the main page, here the user can activate or deactivate, delete or rename the alarm.

Settings

On the settings screen the user can set preferences such as the way the alarm is turned off, snooze time, different alarm sounds.

Alarm rings

When the alarm rings the app is pushed into foreground, the user disables it in his or her prefered way or snoozes it.
